JOSSER

This is the home page for the JOSSER project. Here you can find links to binaries, sources, HOWTOs and more.

JOSSER is a Java library which allows to parse the latest freely available RDF files of DMOZ, the Open Directory Project, and inserts them into any JDBC compliant relational database (i.e. MySQL, PostgreSQL and others to come like Oracle, MS Access, SQLite).
